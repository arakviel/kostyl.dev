# Workflow: Create Educational Material

Цей воркфлоу допоможе вам (або іншому агенту) створити якісну навчальну статтю, дотримуючись менторських принципів.

## Крок 1: Підготовка та Дослідження
1. Визначте тему та цільову аудиторію.
2. Проведіть дослідження (use `context7` mcp) для збору актуальної інформації та прикладів коду.
3. Визначте "проблему" (Hook), з якої почнеться стаття.

## Крок 2: Написання структури (Drafting)
1. Створіть новий `.md` файл у директорії `content/`.
2. Використовуйте `SKILL.md` скіла `material-creator` для дотримання стилю.
3. Сформуйте вступ з фокусом на "Навіщо це потрібно".

## Крок 3: Наповнення контентом та Код
1. Дотримуйтесь правила **"Анатомія Коду"** для кожного лістинга.
2. Використовуйте `::steps` для практичних інструкцій.
3. Додайте візуалізацію:
   - `::mermaid` для схем.
   - `::html-preview` для живих прикладів (якщо це HTML/CSS).
4. Переконайтеся, що текст переважає над кодом (Text First).

## Крок 4: Перевірка та Валідація
1. Перевірте MDC синтаксис за допомогою `resources/DOCUS_COMPONENTS.md`.
2. Переконайтеся, що в кінці є завдання 3-х рівнів складності.
3. Додайте посилання на офіційну документацію.

## Крок 5: Остаточне форматування
1. Перевірте відсутність пробілів перед закриваючими тегами `::` (проблема автоформатування).
2. Надайте матеріалу фінального вигляду.
