| `--`     | –î–µ–∫—Ä–µ–º–µ–Ω—Ç (–∑–º–µ–Ω—à–µ–Ω–Ω—è)  | –ó–º–µ–Ω—à—É—î –∑–Ω–∞—á–µ–Ω–Ω—è –Ω–∞ 1             | `x--` –∞–±–æ `--x` | -         |
| `+`      | –£–Ω–∞—Ä–Ω–∏–π –ø–ª—é—Å           | –í–∫–∞–∑—É—î –ø–æ–∑–∏—Ç–∏–≤–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è         | `+5`            | `5`       |
| `-`      | –£–Ω–∞—Ä–Ω–∏–π –º—ñ–Ω—É—Å          | –ó–º—ñ–Ω—é—î –∑–Ω–∞–∫ –Ω–∞ –ø—Ä–æ—Ç–∏–ª–µ–∂–Ω–∏–π        | `-5`            | `-5`      |

```csharp showLineNumbers
int b = 3;

int sum = a + b;          // 13
int difference = a - b;   // 7

// –î—ñ–ª–µ–Ω–Ω—è –∑ –¥—Ä–æ–±–æ–≤–æ—é —á–∞—Å—Ç–∏–Ω–æ—é

// –£–Ω–∞—Ä–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏
```

#### –Ü–Ω–∫—Ä–µ–º–µ–Ω—Ç —Ç–∞ –¥–µ–∫—Ä–µ–º–µ–Ω—Ç

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ `++` —Ç–∞ `--` –º–æ–∂—É—Ç—å –±—É—Ç–∏ **–ø—Ä–µ—Ñ—ñ–∫—Å–Ω–∏–º–∏** (–ø–µ—Ä–µ–¥ –∑–º—ñ–Ω–Ω–æ—é) –∞–±–æ **–ø–æ—Å—Ç—Ñ—ñ–∫—Å–Ω–∏–º–∏** (–ø—ñ—Å–ª—è –∑–º—ñ–Ω–Ω–æ—ó), —â–æ –≤–ø–ª–∏–≤–∞—î –Ω–∞ –ø–æ—Ä—è–¥–æ–∫ –≤–∏–∫–æ–Ω–∞–Ω–Ω—è.

::code-group

```csharp [–ü—Ä–µ—Ñ—ñ–∫—Å–Ω–∏–π (++x)]

int y = ++x;  // –°–ø–æ—á–∞—Ç–∫—É –∑–±—ñ–ª—å—à–µ–Ω–Ω—è, –ø–æ—Ç—ñ–º –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è

// –í–∏–≤–µ–¥–µ: x = 6, y = 6
```


::

### –û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è (Assignment Operators) –≤—Å—Ç–∞–Ω–æ–≤–ª—é—é—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è –∑–º—ñ–Ω–Ω–æ—ó.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–∞–∑–≤–∞                       | –ï–∫–≤—ñ–≤–∞–ª–µ–Ω—Ç   | –ü—Ä–∏–∫–ª–∞–¥   | –†–µ–∑—É–ª—å—Ç–∞—Ç (—è–∫—â–æ x=5) |
| :------- | :-------------------------- | :----------- | :-------- | :------------------- |
| `=`      | –ü—Ä–æ—Å—Ç–µ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è         | -            | `x = 10`  | `x = 10`             |
| `+=`     | –î–æ–¥–∞–≤–∞–Ω–Ω—è –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º   | `x = x + y`  | `x += 3`  | `x = 8`              |
| `-=`     | –í—ñ–¥–Ω—ñ–º–∞–Ω–Ω—è –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º  | `x = x - y`  | `x -= 3`  | `x = 2`              |
| `*=`     | –ú–Ω–æ–∂–µ–Ω–Ω—è –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º    | `x = x * y`  | `x *= 3`  | `x = 15`             |
| `/=`     | –î—ñ–ª–µ–Ω–Ω—è –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º     | `x = x / y`  | `x /= 2`  | `x = 2`              |
| `%=`     | –û—Å—Ç–∞—á–∞ –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º      | `x = x % y`  | `x %= 3`  | `x = 2`              |
| `&=`     | –ë—ñ—Ç–æ–≤–µ –Ü –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º    | `x = x & y`  | `x &= 3`  | -                    |
| `\|=`    | –ë—ñ—Ç–æ–≤–µ –ê–ë–û –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º  | `x = x \| y` | `x \|= 3` | -                    |
| `^=`     | –ë—ñ—Ç–æ–≤–µ XOR –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º  | `x = x ^ y`  | `x ^= 3`  | -                    |
| `<<=`    | –ó—Å—É–≤ –≤–ª—ñ–≤–æ –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º  | `x = x << y` | `x <<= 2` | `x = 20`             |
| `>>=`    | –ó—Å—É–≤ –≤–ø—Ä–∞–≤–æ –∑ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è–º | `x = x >> y` | `x >>= 1` | `x = 2`              |
| `??=`    | Null-coalescing assignment  | `x = x ?? y` | `x ??= 5` | -                    |

```csharp showLineNumbers
int counter = 0;

// –°–∫–ª–∞–¥–µ–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—é–≤–∞–Ω–Ω—è
counter += 5;   // counter = 0 + 5 = 5
counter *= 2;   // counter = 5 * 2 = 10
counter -= 3;   // counter = 10 - 3 = 7

counter %= 2;   // counter = 3 % 2 = 1

// Null-coalescing assignment (C# 8.0+)
string? name = null;
name ??= "–ó–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º";  // –ü—Ä–∏—Å–≤–æ—é—î —Ç—ñ–ª—å–∫–∏ —è–∫—â–æ name == null
Console.WriteLine(name);       // "–ó–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º"

name ??= "–Ü–Ω—à–µ –∑–Ω–∞—á–µ–Ω–Ω—è";      // –ù–µ –ø—Ä–∏—Å–≤–æ—é—î, –±–æ name –≤–∂–µ –Ω–µ null
Console.WriteLine(name);       // "–ó–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º"
```

### –û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è (Comparison/Relational Operators) –ø–æ—Ä—ñ–≤–Ω—é—é—Ç—å –¥–≤–∞ –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∞ –ø–æ–≤–µ—Ä—Ç–∞—é—Ç—å `bool`.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–∞–∑–≤–∞               | –û–ø–∏—Å                      | –ü—Ä–∏–∫–ª–∞–¥  | –†–µ–∑—É–ª—å—Ç–∞—Ç |
| :------- | :------------------ | :------------------------ | :------- | :-------- |
| `==`     | –î–æ—Ä—ñ–≤–Ω—é—î            | –ü–µ—Ä–µ–≤—ñ—Ä—è—î —Ä—ñ–≤–Ω—ñ—Å—Ç—å        | `5 == 3` | `false`   |
| `!=`     | –ù–µ –¥–æ—Ä—ñ–≤–Ω—é—î         | –ü–µ—Ä–µ–≤—ñ—Ä—è—î –Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å      | `5 != 3` | `true`    |
| `>`      | –ë—ñ–ª—å—à–µ              | –ü–µ—Ä—à–∏–π –±—ñ–ª—å—à–∏–π –∑–∞ –¥—Ä—É–≥–∏–π  | `5 > 3`  | `true`    |
| `<`      | –ú–µ–Ω—à–µ               | –ü–µ—Ä—à–∏–π –º–µ–Ω—à–∏–π –∑–∞ –¥—Ä—É–≥–∏–π   | `5 < 3`  | `false`   |
| `>=`     | –ë—ñ–ª—å—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î | –ü–µ—Ä—à–∏–π –±—ñ–ª—å—à–∏–π –∞–±–æ —Ä—ñ–≤–Ω–∏–π | `5 >= 5` | `true`    |
| `<=`     | –ú–µ–Ω—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î  | –ü–µ—Ä—à–∏–π –º–µ–Ω—à–∏–π –∞–±–æ —Ä—ñ–≤–Ω–∏–π  | `5 <= 3` | `false`   |

```csharp showLineNumbers
int x = 10;
int y = 20;

bool isEqual = x == y;          // false
bool isNotEqual = x != y;       // true
bool isGreater = x > y;         // false
bool isLess = x < y;            // true
bool isGreaterOrEqual = x >= 10; // true
bool isLessOrEqual = x <= 5;    // false

// –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è —Ä—è–¥–∫—ñ–≤
string str1 = "Hello";
string str2 = "World";
bool areStringsEqual = str1 == str2;  // false

// –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –¥–∞—Ç

```

::warning
**–£–≤–∞–≥–∞ –∑ reference types**: –û–ø–µ—Ä–∞—Ç–æ—Ä `==` –¥–ª—è reference types –ø–æ—Ä—ñ–≤–Ω—é—î **–ø–æ—Å–∏–ª–∞–Ω–Ω—è**, –∞ –Ω–µ –≤–º—ñ—Å—Ç –æ–±'—î–∫—Ç—ñ–≤ (–æ–∫—Ä—ñ–º `string`, —è–∫–∏–π –º–∞—î –ø–µ—Ä–µ–≤–∏–∑–Ω–∞—á–µ–Ω—É –ø–æ–≤–µ–¥—ñ–Ω–∫—É). –î–ª—è –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –≤–º—ñ—Å—Ç—É –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ –º–µ—Ç–æ–¥ `.Equals()`.
::

### –õ–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏

–õ–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ (Logical Operators) –≤–∏–∫–æ–Ω—É—é—Ç—å –ª–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó –Ω–∞–¥ –±—É–ª–µ–≤–∏–º–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º–∏.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–∞–∑–≤–∞                         | –û–ø–∏—Å                            | –ü—Ä–∏–∫–ª–∞–¥           | –†–µ–∑—É–ª—å—Ç–∞—Ç |
| :------- | :---------------------------- | :------------------------------ | :---------------- | :-------- |
| `&&`     | –õ–æ–≥—ñ—á–Ω–µ –Ü (AND)               | `true` —è–∫—â–æ –æ–±–∏–¥–≤–∞ `true`       | `true && false`   | `false`   |
| `\|\|`   | –õ–æ–≥—ñ—á–Ω–µ –ê–ë–û (OR)              | `true` —è–∫—â–æ —Ö–æ—á–∞ –± –æ–¥–∏–Ω `true`  | `true \|\| false` | `true`    |
| `!`      | –õ–æ–≥—ñ—á–Ω–µ –ù–ï (NOT)              | –Ü–Ω–≤–µ—Ä—Ç—É—î –±—É–ª–µ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è        | `!true`           | `false`   |
| `&`      | –õ–æ–≥—ñ—á–Ω–µ –Ü –±–µ–∑ short-circuit   | –ó–∞–≤–∂–¥–∏ –æ–±—á–∏—Å–ª—é—î –æ–±–∏–¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∏ | `true & false`    | `false`   |
| `\|`     | –õ–æ–≥—ñ—á–Ω–µ –ê–ë–û –±–µ–∑ short-circuit | –ó–∞–≤–∂–¥–∏ –æ–±—á–∏—Å–ª—é—î –æ–±–∏–¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∏ | `true \| false`   | `true`    |
| `^`      | –õ–æ–≥—ñ—á–Ω–µ –≤–∏–∫–ª—é—á–Ω–µ –ê–ë–û (XOR)    | `true` —è–∫—â–æ –æ–ø–µ—Ä–∞–Ω–¥–∏ —Ä—ñ–∑–Ω—ñ      | `true ^ false`    | `true`    |

```csharp showLineNumbers
bool isAdult = true;
bool hasLicense = false;

// –õ–æ–≥—ñ—á–Ω–µ –Ü (AND) - –æ–±–∏–¥–≤–∞ –º–∞—é—Ç—å –±—É—Ç–∏ true

// –õ–æ–≥—ñ—á–Ω–µ –ê–ë–û (OR) - —Ö–æ—á–∞ –± –æ–¥–∏–Ω true

// –õ–æ–≥—ñ—á–Ω–µ –ù–ï (NOT) - —ñ–Ω–≤–µ—Ä—Å—ñ—è

// –°–∫–ª–∞–¥–Ω—ñ —É–º–æ–≤–∏
int age = 25;
bool hasPermission = true;

if (age >= 18 && hasPermission)
{
    Console.WriteLine("–î–æ—Å—Ç—É–ø –¥–æ–∑–≤–æ–ª–µ–Ω–æ");
}

// XOR - –≤–∏–∫–ª—é—á–Ω–µ –ê–ë–û
bool option1 = true;
bool option2 = false;
bool exclusiveChoice = option1 ^ option2;  // true (—Ç—ñ–ª—å–∫–∏ –æ–¥–∏–Ω –∑ –Ω–∏—Ö true)
```

#### Short-circuit evaluation

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ `&&` —Ç–∞ `||` –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å **short-circuit evaluation** (—Å–∫–æ—Ä–æ—á–µ–Ω–µ –æ–±—á–∏—Å–ª–µ–Ω–Ω—è) ‚Äî –¥—Ä—É–≥–∏–π –æ–ø–µ—Ä–∞–Ω–¥ –Ω–µ –æ–±—á–∏—Å–ª—é—î—Ç—å—Å—è, —è–∫—â–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤–∂–µ –≤—ñ–¥–æ–º–∏–π –∑ –ø–µ—Ä—à–æ–≥–æ.

```csharp showLineNumbers
int? nullableValue = null;

// && short-circuit: –¥—Ä—É–≥–∏–π –≤–∏—Ä–∞–∑ –Ω–µ –≤–∏–∫–æ–Ω–∞—î—Ç—å—Å—è, —è–∫—â–æ –ø–µ—Ä—à–∏–π false
if (nullableValue != null && nullableValue.Value > 10) // [!code highlight]
{
    // –ë–µ–∑–ø–µ—á–Ω–æ: nullableValue.Value –Ω–µ –≤–∏–∫–ª–∏—á–µ—Ç—å—Å—è, —è–∫—â–æ nullableValue == null
}

// || short-circuit: –¥—Ä—É–≥–∏–π –≤–∏—Ä–∞–∑ –Ω–µ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è, —è–∫—â–æ –ø–µ—Ä—à–∏–π true
bool result = CheckCondition1() || CheckCondition2(); // [!code highlight]
// –Ø–∫—â–æ CheckCondition1() –ø–æ–≤–µ—Ä—Ç–∞—î true, CheckCondition2() –ù–ï –≤–∏–∫–ª–∏–∫–∞—î—Ç—å—Å—è

// –ë–µ–∑ short-circuit (& —Ç–∞ |) - –æ–±–∏–¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∏ –∑–∞–≤–∂–¥–∏ –æ–±—á–∏—Å–ª—é—é—Ç—å—Å—è
bool result2 = CheckCondition1() & CheckCondition2(); // [!code highlight]
// –û–±–∏–¥–≤–∞ –º–µ—Ç–æ–¥–∏ –ó–ê–í–ñ–î–ò –≤–∏–∫–ª–∏–∫–∞—é—Ç—å—Å—è, –Ω–µ–∑–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É –ø–µ—Ä—à–æ–≥–æ
```

::tip
**Best Practice**: –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ `&&` —Ç–∞ `||` –∑–∞–º—ñ—Å—Ç—å `&` —Ç–∞ `|` –¥–ª—è –±—É–ª–µ–≤–∏—Ö –æ–ø–µ—Ä–∞—Ü—ñ–π, –æ—Å–∫—ñ–ª—å–∫–∏ short-circuit evaluation:

-   –ü—ñ–¥–≤–∏—â—É—î –ø—Ä–æ–¥—É–∫—Ç–∏–≤–Ω—ñ—Å—Ç—å (–Ω–µ –æ–±—á–∏—Å–ª—é—î –Ω–µ–ø–æ—Ç—Ä—ñ–±–Ω—ñ –≤–∏—Ä–∞–∑–∏)
-   –ó–∞–ø–æ–±—ñ–≥–∞—î –ø–æ–º–∏–ª–∫–∞–º (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `NullReferenceException`)

::

### –¢–µ—Ä–Ω–∞—Ä–Ω–∏–π –æ–ø–µ—Ä–∞—Ç–æ—Ä

–¢–µ—Ä–Ω–∞—Ä–Ω–∏–π (—É–º–æ–≤–Ω–∏–π) –æ–ø–µ—Ä–∞—Ç–æ—Ä `? :` (Ternary/Conditional Operator) ‚Äî —Ü–µ —Å–∫–æ—Ä–æ—á–µ–Ω–∞ —Ñ–æ—Ä–º–∞ `if-else`.

**–°–∏–Ω—Ç–∞–∫—Å–∏—Å:**

```csharp

```

```csharp showLineNumbers
int age = 20;

// –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è if-else

if (age >= 18)
{
    status = "–ü–æ–≤–Ω–æ–ª—ñ—Ç–Ω—ñ–π";
}
else
{
    status = "–ù–µ–ø–æ–≤–Ω–æ–ª—ñ—Ç–Ω—ñ–π";
}

// –¢–µ —Å–∞–º–µ –∑ —Ç–µ—Ä–Ω–∞—Ä–Ω–∏–º –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º
```

## Value Types vs Reference Types

–û–¥–Ω–∞ –∑ –Ω–∞–π–≤–∞–∂–ª–∏–≤—ñ—à–∏—Ö –∫–æ–Ω—Ü–µ–ø—Ü—ñ–π —É C# ‚Äî —Ü–µ —Ä—ñ–∑–Ω–∏—Ü—è –º—ñ–∂ —Ç–∏–ø–∞–º–∏ –∑–Ω–∞—á–µ–Ω—å (Value Types) —Ç–∞ —Ç–∏–ø–∞–º–∏ –ø–æ—Å–∏–ª–∞–Ω—å (Reference Types).

### –¢–∏–ø–∏ –∑–Ω–∞—á–µ–Ω—å (Value Types)

–¢–∏–ø–∏ –∑–Ω–∞—á–µ–Ω—å –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å –¥–∞–Ω—ñ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ. –í–æ–Ω–∏ —Ä–æ–∑–º—ñ—â—É—é—Ç—å—Å—è –≤ **—Å—Ç–µ–∫—É** (Stack) –∞–±–æ inline –≤ –æ–±'—î–∫—Ç—ñ, —â–æ —ó—Ö –º—ñ—Å—Ç–∏—Ç—å.

**–¢–∏–ø–∏ –∑–Ω–∞—á–µ–Ω—å –≤–∫–ª—é—á–∞—é—Ç—å:**

-   –£—Å—ñ —á–∏—Å–ª–æ–≤—ñ —Ç–∏–ø–∏ (`int`, `double`, `decimal` —Ç–æ—â–æ)
-   `bool`, `char`
-   `struct`
-   `enum`
-   Nullable value types (`int?`, `bool?` —Ç–æ—â–æ)

```csharp {2-3}
// Value types - –∫–æ–ø—ñ—é–≤–∞–Ω–Ω—è –∑–Ω–∞—á–µ–Ω–Ω—è
int a = 10;
int b = a;    // –ö–æ–ø—ñ—é—î—Ç—å—Å—è –∑–Ω–∞—á–µ–Ω–Ω—è // [!code highlight]
b = 20;       // –ó–º—ñ–Ω—é—î—Ç—å—Å—è —Ç—ñ–ª—å–∫–∏ b // [!code highlight]

Console.WriteLine($"a = {a}, b = {b}"); // a = 10, b = 20
```

### Boxing —Ç–∞ Unboxing

**Boxing** ‚Äî —Ü–µ –ø—Ä–æ—Ü–µ—Å –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è value type —É reference type (–∑–∞–∑–≤–∏—á–∞–π –≤ `object`).

## –ë—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó

–ë—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó (Bitwise Operations) –¥–æ–∑–≤–æ–ª—è—é—Ç—å –º–∞–Ω—ñ–ø—É–ª—é–≤–∞—Ç–∏ –æ–∫—Ä–µ–º–∏–º–∏ –±—ñ—Ç–∞–º–∏ –≤ —á–∏—Å–ª–∞—Ö. –í–æ–Ω–∏ —á–∞—Å—Ç–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –æ–ø—Ç–∏–º—ñ–∑–∞—Ü—ñ—ó –ø–∞–º'—è—Ç—ñ, —Ä–æ–±–æ—Ç–∏ –∑ –ø—Ä–∞–ø–æ—Ä—Ü—è–º–∏ —Ç–∞ –Ω–∏–∑—å–∫–æ—Ä—ñ–≤–Ω–µ–≤–æ–≥–æ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è.

### –ë—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–∞–∑–≤–∞       | –û–ø–∏—Å                           | –ü—Ä–∏–∫–ª–∞–¥         |
