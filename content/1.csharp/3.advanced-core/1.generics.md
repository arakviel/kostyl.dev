### Covariance (`out` keyword)

[**Covariance**](https://learn.microsoft.com/en-us/dotnet/csharp/programming-guide/concepts/covariance-contravariance/covariance-and-contravariance-in-generics) –¥–æ–∑–≤–æ–ª—è—î –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ **–±—ñ–ª—å—à –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏–π** (derived) —Ç–∏–ø –∑–∞–º—ñ—Å—Ç—å **–∑–∞–≥–∞–ª—å–Ω–æ–≥–æ** (base).

```csharp showLineNumbers
// IEnumerable<T> —î covariant (–º–∞—î out modifier)

{
    new Dog { Name = "Rex" },
    new Dog { Name = "Buddy" }
};

// ‚úÖ Covariance: Dog ‚Üí Animal

{
    Console.WriteLine(animal.Name); // Rex, Buddy
}
```

**–ß–æ–º—É —Ü–µ –ø—Ä–∞—Ü—é—î?**

-   `Dog` —î `Animal`
-   –ú–∏ **—Ç—ñ–ª—å–∫–∏ —á–∏—Ç–∞—î–º–æ** –∑ `IEnumerable<T>` (get, iterate)
-   –ë–µ–∑–ø–µ—á–Ω–æ –ø–æ–≤–µ—Ä—Ç–∞—Ç–∏ `Dog` –∫–æ–ª–∏ –æ—á—ñ–∫—É—î—Ç—å—Å—è `Animal`

**–í–∏–∑–Ω–∞—á–µ–Ω–Ω—è Covariant Interface:**

**–¢–µ–æ—Ä—ñ—è Covariance:**

Covariance –¥–æ–∑–≤–æ–ª—è—î –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –±—ñ–ª—å—à –ø–æ—Ö—ñ–¥–Ω–∏–π —Ç–∏–ø –Ω—ñ–∂ –≤–∫–∞–∑–∞–Ω–∏–π, —Ç–æ–±—Ç–æ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ `IEnumerable<Dog>` —Ç–∞–º, –¥–µ –æ—á—ñ–∫—É—î—Ç—å—Å—è `IEnumerable<Animal>`. –¶–µ –º–æ–∂–ª–∏–≤–æ –ª–∏—à–µ –¥–ª—è –æ–ø–µ—Ä–∞—Ü—ñ–π, —â–æ –ø–æ–≤–µ—Ä—Ç–∞—é—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è (output positions), –æ—Å–∫—ñ–ª—å–∫–∏ –º–∏ –≥–∞—Ä–∞–Ω—Ç–æ–≤–∞–Ω–æ –ø–æ–≤–µ—Ä—Ç–∞—î–º–æ —Ç–∏–ø, —Å—É–º—ñ—Å–Ω–∏–π –∑ –æ—á—ñ–∫—É–≤–∞–Ω–∏–º.

**–í–Ω—É—Ç—Ä—ñ—à–Ω—ñ –º–µ—Ö–∞–Ω—ñ–∑–º–∏ covariance:**

-   –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –º–æ–¥–∏—Ñ—ñ–∫–∞—Ç–æ—Ä `out` —É –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—ñ —ñ–Ω—Ç–µ—Ä—Ñ–µ–π—Å—É –∞–±–æ –¥–µ–ª–µ–≥–∞—Ç–∞
-   –ó–∞—Å—Ç–æ—Å–æ–≤—É—î—Ç—å—Å—è –ª–∏—à–µ –¥–æ –ø–∞—Ä–∞–º–µ—Ç—Ä—ñ–≤ —Ç–∏–ø—É, —â–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –≤ –ø–æ–∑–∏—Ü—ñ—è—Ö "–Ω–∞ –≤–∏—Ö—ñ–¥" (return types, –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ —Ç—ñ–ª—å–∫–∏ –¥–ª—è —á–∏—Ç–∞–Ω–Ω—è)

### Contravariance (`in` keyword)

[**Contravariance**](https://learn.microsoft.com/en-us/dotnet/csharp/programming-guide/concepts/covariance-contravariance/covariance-and-contravariance-in-generics) –¥–æ–∑–≤–æ–ª—è—î –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ **–±—ñ–ª—å—à –∑–∞–≥–∞–ª—å–Ω–∏–π** (base) —Ç–∏–ø –∑–∞–º—ñ—Å—Ç—å **–∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ** (derived).

```csharp showLineNumbers
// IComparer<T> —î contravariant (–º–∞—î in modifier)

{
    public int Compare(Animal x, Animal y)
    {
        return string.Compare(x.Name, y.Name);
    }
}

// ‚úÖ Contravariance: Animal ‚Üí Dog

{
    new Dog { Name = "Buddy" },
    new Dog { Name = "Rex" }
```

**–ß–æ–º—É —Ü–µ –ø—Ä–∞—Ü—é—î?**

-   –ú–µ—Ç–æ–¥ –ø—Ä–∏–π–º–∞—î `Animal`, —Ç–æ–º—É –º–æ–∂–µ –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –∑ `Dog`
-   –ú–∏ **—Ç—ñ–ª—å–∫–∏ –ø–∏—à–µ–º–æ** –≤ `IComparer<T>` (input parameters)
-   –ë–µ–∑–ø–µ—á–Ω–æ –ø–µ—Ä–µ–¥–∞–≤–∞—Ç–∏ `Dog` –∫–æ–ª–∏ –æ—á—ñ–∫—É—î—Ç—å—Å—è `Animal`

**–í–Ω—É—Ç—Ä—ñ—à–Ω—ñ –º–µ—Ö–∞–Ω—ñ–∑–º–∏ contravariance:**

-   –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –º–æ–¥–∏—Ñ—ñ–∫–∞—Ç–æ—Ä `in` —É –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—ñ —ñ–Ω—Ç–µ—Ä—Ñ–µ–π—Å—É –∞–±–æ –¥–µ–ª–µ–≥–∞—Ç–∞

-   –ü—Ä–∞—Ü—é—î –Ω–∞ —Ä—ñ–≤–Ω—ñ –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä–∞ - –ø—ñ–¥ —á–∞—Å –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó –ø–µ—Ä–µ–≤—ñ—Ä—è—î—Ç—å—Å—è —Å—É–º—ñ—Å–Ω—ñ—Å—Ç—å —Ç–∏–ø—ñ–≤

### Invariance (–ó–∞ –ó–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º)

–Ø–∫—â–æ –Ω–µ–º–∞—î `in` –∞–±–æ `out`, generic —Ç–∏–ø —î **invariant** ‚Äî –Ω–µ –º–æ–∂–Ω–∞ –∑–∞–º—ñ–Ω—é–≤–∞—Ç–∏ —Ç–∏–ø–∏:

```csharp showLineNumbers

    T GetById(int id);     // Output

// IRepository<Animal> animalRepo = dogRepo; // ‚ùå –ù–µ –ø—Ä–∞—Ü—é—î ‚Äî invariant
```

Invariance –æ–∑–Ω–∞—á–∞—î, —â–æ `IRepository<Dog>` —ñ `IRepository<Animal>` —î –∞–±—Å–æ–ª—é—Ç–Ω–æ —Ä—ñ–∑–Ω–∏–º–∏ —Ç–∏–ø–∞–º–∏, –Ω–∞–≤—ñ—Ç—å —è–∫—â–æ –º—ñ–∂ `Dog` —ñ `Animal` —ñ—Å–Ω—É—î –≤—ñ–¥–Ω–æ—à–µ–Ω–Ω—è —É—Å–ø–∞–¥–∫—É–≤–∞–Ω–Ω—è. –¶–µ –Ω–∞–π–±–µ–∑–ø–µ—á–Ω—ñ—à–∏–π –≤–∞—Ä—ñ–∞–Ω—Ç, –æ—Å–∫—ñ–ª—å–∫–∏ –∑–∞–±–æ—Ä–æ–Ω—è—î –±—É–¥—å-—è–∫—ñ –ø–æ—Ç–µ–Ω—Ü—ñ–π–Ω–æ –Ω–µ–±–µ–∑–ø–µ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó –∑ —Ç–∏–ø–∞–º–∏.

-   –ó–∞–ø–æ–±—ñ–≥–∞—î –ø–æ–º–∏–ª–∫–∞–º —Ç–∏–ø—ñ–≤ –ø—ñ–¥ —á–∞—Å –≤–∏–∫–æ–Ω–∞–Ω–Ω—è

```csharp showLineNumbers
{
```

```csharp showLineNumbers
{
```

```csharp showLineNumbers
    private int _nextId = 1;

    public void Add(T item)
    {
        item.Id = _nextId++;
        _items.Add(item);
    }

    public T GetById(int id)
    {
        return _items.FirstOrDefault(x => x.Id == id);
    }

    public IEnumerable<T> GetAll()
    {
        return _items.ToList();
    }

            _items[index] = item;
        }
    }

    {
        var item = GetById(id);
        if (item != null)
        {
            _items.Remove(item);
        }
    }
}
```

```csharp showLineNumbers
    }

    public T Create()
    {
        if (_isSingleton)
        {
            if (_cachedInstance == null)
            {
                _cachedInstance = new T();
            }
            return _cachedInstance;
        }

    }
}
```

```csharp showLineNumbers
    IWriteOnlyAnimalCollection<T>

}
```
