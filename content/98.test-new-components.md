## WpfPreview (Avalonia WASM Demo)

Цей компонент створений для імітації роботи справжнього XAML рушія у браузері.
Він показує анімацію завантаження WASM-модуля і рендерить візуальний "Canvas" з контролерами у стилі Windows 11.

Оскільки під капотом працює **Avalonia 11**, вона вже включає в себе **Fluent Theme** (Windows 11). Тому вам не потрібно вручну фарбувати кнопки — достатньо використовувати вбудовані класи (як от `Classes="accent"`) і все матиме нативий вигляд!

::wpf-preview{title="WPF Preview - Dynamic Demo"}

```xml
<UserControl xmlns="https://github.com/avaloniaui"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <StackPanel Margin="24" Spacing="16">
        <TextBlock Text="Динамічний інтерфейс (Data Binding)" FontSize="20" FontWeight="Bold" />
        
        <!-- Це поле оновиться динамічно через Binding -->
        <Border Background="{DynamicResource SystemAccentColorLight3}" 
                CornerRadius="8" Padding="12">
            <StackPanel Orientation="Horizontal" HorizontalAlignment="Center">
                <TextBlock Text="Значення лічильника: " VerticalAlignment="Center" />
                <TextBlock Text="{Binding Counter}" FontSize="24" FontWeight="Black" 
                           Foreground="{DynamicResource SystemAccentColor}"
                           Margin="8,0,0,0" />
            </StackPanel>
        </Border>

        <!-- Кнопка, що змінює стан ViewModel -->
        <Button Content="Збільшити число (+1)" 
                Command="{Binding IncrementCounterCommand}" 
                Classes="accent" 
                HorizontalAlignment="Stretch" 
                HorizontalContentAlignment="Center" />

        <Separator Margin="0,10" />

        <Button Content="Надіслати лог" 
                Command="{Binding ShowMessageCommand}" 
                CommandParameter="Лог надіслано з динамічного вікна!" 
                HorizontalAlignment="Center" />
    </StackPanel>
</UserControl>
```

```csharp
using AvaloniaHost.ViewModels;

namespace AvaloniaHost;

public partial class MainWindow : Window
{
    // У нас є властивість Counter (int)
    // Та команда IncrementCounterCommand
    
    public void OnIncrement()
    {
        // Це те, що відбувається під капотом
        var vm = MainViewModel.Instance;
        vm.Counter++; 
        
        // Avalonia сама оновить UI через Binding
    }
}
```
