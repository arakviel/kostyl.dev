# Персональний Бюджет-Трекер

Консольний додаток для відстеження персональних фінансів, що демонструє основні концепції роботи зі змінними та типами даних у C#.

## Що демонструє проєкт

-   **Змінні та константи**: `const`, `readonly`, `static readonly`
-   **Типи даних**: `int`, `decimal`, `string`, `bool`, `enum`, `DateTime`
-   **Оператори**: арифметичні, порівняння, логічні, бітові
-   **Конверсія типів**: `TryParse`, явні приведення
-   **Value vs Reference Types**: `struct` (Transaction) vs `class` (Budget)
-   **Nullable types**: робота з `null`
-   **Бітові операції**: `[Flags]` enum для категорій
-   **Правила іменування**: camelCase, PascalCase, \_camelCase

## Запуск проєкту

```bash
dotnet run
```

## Функціональність

1. Додавання доходів та витрат
2. Перегляд транзакцій з фільтрацією
3. Статистика бюджету
4. Конвертація валют
5. Демонстрації концепцій C#
